## Run Locally

Clone the project

```bash
  git clone https://github.com/shubhkhanna/binks-assignment.git
```

Go to the project directory

```bash
  cd binks-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
